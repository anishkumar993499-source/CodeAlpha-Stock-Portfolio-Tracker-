Stock Portfolio Tracker

A simple Python project to track stock investments and calculate the total portfolio value.

Features

- Add stock names
- Enter quantity of shares
- Enter stock prices
- Calculate total investment value
- Display portfolio summary
- Uses loops and conditional statements
- Beginner-friendly Python project

Requirements

- Python 3

Concepts Used

- Variables
- Lists
- Dictionaries (optional)
- Loops ("while", "for")
- Conditional Statements ("if", "elif", "else")
- User Input ("input()")
- Arithmetic Operations

How to Run

1. Save the program as "stock_tracker.py"
2. Open Terminal or Command Prompt
3. Run:

python stock_tracker.py

Example

=== STOCK PORTFOLIO TRACKER ===

Enter Stock Name: AAPL
Enter Quantity: 10
Enter Price per Share: 200

Stock: AAPL
Quantity: 10
Price: 200

Total Value: 2000
